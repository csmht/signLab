# 题目答案接口契约

## 1. 适用范围

本契约适用于以下接口中的题目答案字段：

- 题库题目创建、更新、导入
- 课堂小测提交与结果查询
- 题库答题步骤提交与结果查询
- 限时答题步骤提交与结果查询

## 2. 标准答案格式

### 单选题

- 标准值：选项字母
- 示例：`A`

### 多选题

- 标准值：使用 `-` 连接的选项字母
- 示例：`A-B-C`

### 判断题

- 标准值：`正确` / `错误`
- 兼容旧值：`A` / `B` / `T` / `F`
- 内部存储：统一归一化为 `T` / `F`
- 接口返回：统一返回 `正确` / `错误`

### 填空题 / 简答题

- 标准值：直接使用原始文本

## 3. 学生提交结构

题库答题、限时答题、课堂小测统一使用：

```json
{
  "answers": [
    { "topicId": 101, "answer": "A" },
    { "topicId": 102, "answer": "A-B-C" },
    { "topicId": 103, "answer": "正确" }
  ]
}
```

## 4. 存储结构

### 题库答题

```json
{
  "type": "TOPIC",
  "data": {
    "101": "A",
    "103": "T"
  }
}
```

### 限时答题

```json
{
  "type": "TIMED_QUIZ",
  "data": {
    "201": "A-B-C",
    "202": "F"
  }
}
```

## 5. 兼容策略

- 写入时：判断题旧值会自动归一化到 `T` / `F`
- 比较时：判断题按语义比较，不再受 `正确/错误`、`A/B`、`T/F` 混用影响
- 返回时：判断题统一返回 `正确` / `错误`
